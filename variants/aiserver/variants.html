Meaning of the files

MDF — Map Definition
These files contain the MDF message sent by the server, in Diplomacy Message syntax.
NAM — Power and Province Names
These files contain full names for each of the powers and provinces on the map, as well as possessives for each power.
NOW — Starting Position
These files contain the NOW message sent at the beginning of the game, in Diplomacy Message syntax.
REM — Representation Message
These files contain token numbers and abbreviations for the Representation Message sent by the server.
SCO — Supply Center Ownership
These files contain the SCO message sent at the beginning of the game, in Diplomacy Message syntax.
TTY — ASCII Map Data
These files contain data for displaying a map on a sixteen-color text terminal. Not nearly as useful, for those of you with graphical operating systems, as the Mapper's files.
